Clearline7 Brand-Aesthetics Asset Manifest

This manifest defines the visual assets (icons, emojis, clip art/illustrations) required for each SetDefinition in the Clearline7 ecosystem. Assets are organized by set and type, with prompt templates included for generation.

Asset Categories

1. Icons (SVG/PNG)

Minimalist flat icons representing key concepts.

Use brand colors only.

Geometric, no gradients.

Prompt Template:

Minimalist flat icon of [concept] in [brand colors], geometric, no gradients.

2. Emoji Equivalents

Unicode emoji mapped to brand concepts.

Used for lightweight communication.

3. Clip Art / Illustrations

Flat illustrations of brand concepts.

Use brand palette.

Clean lines, no shading.

Prompt Template:

Flat illustration of [concept] using [brand palette], clean lines, no shading.
