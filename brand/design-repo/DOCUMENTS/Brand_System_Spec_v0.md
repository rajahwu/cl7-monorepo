# Clearline7 Brand System Spec (v0)

## 1. Brand Essence

**Positioning:** Confident Expert.
**Promise:** Every document looks like it came from the same competent mind.

## 2. Voice and Tone

- **Voice:** Clear, direct, concise. No hedging.
- **Do:** "Use this format."
- **Don't:** "We kind of prefer..."

## 3. Color System

- **Deep Blue (#1E3A8A):** Authority.
- **Cool Gray (#6B7280):** Structure.
- **Soft Teal (#14B8A6):** Governed Flexibility.

## 4. Typography

- **Headings:** Modern Sans-Serif (Inter), Bold.
- **Body:** Regular weight, high readability.
