Asset Naming and Usage Conventions

This document defines the naming rules and usage guidelines for all visual assets in the Clearline7 brand-aesthetic system.

1. Icons

Format: .svg

Style: Flat, geometric, no gradients

Naming: icon*[concept]*[set].svg

Example: icon_box_boxkit.svg

2. Emoji Equivalents

Format: .png

Purpose: Lightweight symbolic communication

Naming: emoji*[concept]*[set].png

Example: emoji_gavel_techlaw.png

3. Clip Art / Illustrations

Format: .png

Style: Flat, clean lines, brand palette only

Naming: clipart*[concept]*[set].png

Example: clipart_dashboard_elementseven.png
